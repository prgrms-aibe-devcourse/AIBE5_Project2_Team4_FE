AIBE5 Team4 Project 2 FrontEnd Github Repository.
초기세팅은 npm을 이용하여 vite@latest 를 설치, 이후 React와 Typescript 기반 프로그램 세팅을 마침.
이후 Svelte사용을 위해 의존성과 Svelte 라이브러리 설치. 이후 브라우저 단에서 프론트 서버로 request 할 시 컴파일 된 파일만 배포 가능. 
이렇게 하면 지연로딩 같은 문제를 상쇄시킬 수 있음.

기술스택
- React, Vite
- Svelte
- Node.js
- Typescript
- ESLint
- CSS
- FFMpeg
